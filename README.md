## 🚀 Как запустить код

### 1. Установка Python
Скачайте и установите Python 3.9 или новее с [официального сайта](https://www.python.org/downloads/)

### 2. Установка необходимых библиотек
Откройте командную строку (Win+R → cmd) и выполните:

```bash
pip install pandas numpy matplotlib seaborn scipy statsmodels opencv-python scikit-learn plotly pymorphy3 wordcloud
```

### 3. Скачивание данных

| Тип данных | Источник | Инструкция |
|------------|----------|------------|
| Табличные | [Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) | Нажмите "Download", файл `creditcard.csv` положите в `data/table/` |
| Временные ряды | [Yahoo Finance](https://finance.yahoo.com/quote/NVDA/history) | Установите период 2016-2026 → Download → файл в `data/timeseries/` |
| Изображения | Собственный датасет | Распределите по папкам `train/helmet/`, `train/no_helmet/`, `test/helmet/`, `test/no_helmet/` |
| Текстовые | Создается автоматически | Код `chapter4_text.py` сам создаст файл `data/text/appeals.csv` |

### 4. Запуск анализа

```bash
# Перейдите в папку с кодом
cd D:/2 семестр/курсач/code

# Запустите анализ по главам
python chapter1_table.py
python chapter2_timeseries.py
python chapter3_images.py
python chapter4_text.py
```

### 5. Результаты
Все рисунки и графики сохранятся в папку `results/`

## 📊 Результаты анализа

### Глава 1: Табличные данные (Credit Card Fraud Detection)
- **Размер датасета:** 284 807 транзакций
- **Мошеннических транзакций:** 492 (0.173%)
- **Ключевые признаки:** V17 (-0.326), V14 (-0.303), V11 (+0.155)
- **Выбросы:** 31 685 (11.17%) в признаке Amount

### Глава 2: Временные ряды (NVIDIA stocks)
- **Период:** 2016–2025 (2514 торговых дней)
- **Рост цены:** с 0.62$ до 207$ (более чем в 330 раз)
- **SNR (отношение сигнал/шум):** 23.26 дБ (отлично)
- **Корреляция цена-объем:** -0.36

### Глава 3: Изображения (Hard Hat Workers)
- **Размер датасета:** 5 000 изображений
- **Баланс классов:** 91.6% (helmet) / 8.4% (no_helmet)
- **Размер изображений:** 415×416 пикселей
- **Выявленные проблемы:** дисбаланс классов, недостаточное разрешение

### Глава 4: Текстовые данные (Обращения граждан)
- **Размер датасета:** 40 обращений, 7 категорий
- **Наиболее частые слова:** улица (36), просить (35), дом (20)
- **Размер словаря TF-IDF:** 200 слов
- **Точность поиска:** похожесть до 0.63
